# Nagham (نَغَم)

> **Significado:** "Melodia" ou "Tom"  
> **Origem:** Árabe antigo

Nagham é uma aplicação web interativa para visualização e estudo de notas musicais e acordes através de três visualizações sincronizadas: partitura (pentagrama), teclado de piano e braço de violão. O projeto foi criado para glorificar a Deus através da música e tem como objetivo evoluir para uma plataforma completa de teoria musical.

## ✨ Características

- **Visualizações Sincronizadas**: Três visualizações interconectadas que destacam a mesma nota quando você clica em qualquer uma delas
  - 📜 **Partitura**: Pentagrama com clave de sol e notas musicais
  - 🎹 **Piano**: Teclado interativo com teclas brancas e pretas
  - 🎸 **Violão**: Braço do violão com cordas, trastes e casas clicáveis

- **Construtor de Acordes**: Interface para construir e visualizar acordes com diferentes qualidades (maior, menor, aumentado, diminuto, etc.)

- **Multilíngue**: Suporte para Português (Brasil), Inglês e Espanhol

- **Tema Escuro**: Interface moderna com tema escuro otimizado para visualização musical

## 🚀 Tecnologias

- **React 19** - Biblioteca UI
- **Vite** - Build tool e dev server
- **D3.js** - Visualizações e gráficos SVG
- **Tailwind CSS 4** - Estilização
- **i18next** - Internacionalização

## 🛠️ Desenvolvimento

```bash
# Instalar dependências
npm install

# Iniciar servidor de desenvolvimento
npm run dev

# Build para produção
npm run build

# Preview da build
npm run preview
```
